# CoinGecko Token Listing Submission Guide

Complete guide for submitting TWUSD token to CoinGecko for listing and logo display.

## üìã Requirements

### Token Requirements
- Token contract deployed and verified
- Token has on-chain activity and liquidity
- Token has at least one DEX pair (e.g., Uniswap, SushiSwap)
- Sufficient trading volume
- Active community and project

### Image Specifications
- **Format**: PNG with transparent background
- **Size**: 200x200 pixels (recommended)
- **Maximum File Size**: 500KB
- **Transparency**: Preferred
- **Aspect Ratio**: 1:1 (square)

### Additional Requirements
- Official project website (recommended)
- Social media presence (Twitter, Telegram, Discord)
- Project documentation
- Whitepaper or litepaper (if applicable)

### Step 2: Submit Request Form

CoinGecko has two submission methods:

#### Method 1: Online Request Form (Recommended)

1. Visit CoinGecko listing request page:
   **[https://www.coingecko.com/en/coins/request](https://www.coingecko.com/en/coins/request)**

2. Fill out the request form:
   - **Project Name**: TheUSDOX Wrapped Dollar EVM
   - **Ticker Symbol**: TWUSD
   - **Project Website**: [Your website URL]
   - **Blockchain/Platform**: Ethereum
   - **Contract Address**: `0x7BeB51807E3c8BdB10A2868bD51c2D9E1764925D`
   - **Description**: Provide detailed project description
   - **Trading Markets**: List all DEX/CEX where TWUSD is traded
   - **Official Links**: Social media, documentation, etc.

3. Upload logo when requested

4. Submit the form

#### Method 2: Email Submission

If the form is unavailable, submit via email:

**To**: tokens@coingecko.com

**Subject**: Token Listing Request - TWUSD (TheUSDOX Wrapped Dollar EVM)

**Email Template**:

```

I would like to request the listing of TWUSD token on CoinGecko.

TOKEN INFORMATION:
- Token Name: TheUSDOX Wrapped Dollar EVM
- Symbol: TWUSD
- Blockchain: Ethereum (Mainnet)
- Contract Address: 0x7BeB51807E3c8BdB10A2868bD51c2D9E1764925D
- Token Type: ERC20
- Decimals: 18

PROJECT INFORMATION:
- Official Website: [Your website]
- Twitter: [Your Twitter]
- Telegram: [Your Telegram]
- Discord: [Your Discord]

- Primary Pair: TWUSD/[pair]

- Etherscan: https://etherscan.io/token/0x7BeB51807E3c8BdB10A2868bD51c2D9E1764925D

[Your Name]
[Your Role]
```

**Attachments**:
- logo.png (200x200 or 256x256)
- Additional documentation (optional)

### Step 3: Provide Additional Information

CoinGecko may request:
- Proof of project legitimacy
- Team information
- Tokenomics details
- Audit reports (if available)
- Partnership information
- Roadmap

Be prepared to provide:
- Project whitepaper or documentation
- Official social media accounts
- Proof of liquidity
- Trading volume data
- Community size

### Step 4: Wait for Review

- **Review Time**: Varies (can be 1-4 weeks or longer)
- **Communication**: Respond promptly to CoinGecko team requests
- **Status**: Check your email for updates
- **Follow-up**: Can follow up after 2 weeks if no response

### Step 5: After Approval

Once approved and listed:
1. Token page will be created on CoinGecko
2. Logo will appear on the token page
3. Price tracking will be enabled
4. Token will appear in search results
5. Historical data will start collecting

## ‚ö†Ô∏è Common Issues & Tips

### Issue: Submission Ignored
**Solutions:**
- Ensure token has real trading activity
- Verify sufficient liquidity exists
- Provide complete information
- Follow up after 2 weeks

### Issue: Incomplete Data
**Solution:** Provide all requested information upfront:
- Website must be professional and informative
- Social media should be active
- Token must have real utility and purpose

### Issue: Logo Not Displayed
**Solution:** 
- Logo must meet specifications (PNG, transparent, square)
- File size under 500KB
- High quality image

### Issue: Low Priority Review
**Solution:**
- Tokens with higher volume/liquidity get priority
- Active communities get faster review
- Professional presentation helps
