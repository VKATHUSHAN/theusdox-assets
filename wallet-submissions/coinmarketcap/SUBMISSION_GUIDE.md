### Step 1: Create CoinMarketCap Account

1. Visit [CoinMarketCap](https://coinmarketcap.com/)
2. Sign up for a free account
3. Verify your email address
4. Complete account setup

### Step 2: Access Self-Service Listing Portal

1. Log in to your CoinMarketCap account
2. Visit the listing portal:
   **[https://coinmarketcap.com/request/](https://coinmarketcap.com/request/)**
3. Click "Add Cryptocurrency" or "Add DeFi Token"

### Step 3: Prepare Your Logo

Use the 256x256px logo (CoinMarketCap accepts it):
```
/branding/twusd/logo-256x256.png
```

Or create 200x200px version:
```bash
# Using ImageMagick
convert logo-256x256.png -resize 200x200 logo-200x200.png
```

Ensure:
- PNG format
- Transparent background
- 200x200 pixels (exact)
- < 500KB file size
- High quality and clear

### Step 5: Upload Logo

1. Click "Upload Logo" in the form
2. Select your prepared logo file (200x200 PNG)
3. Verify logo appears correctly in preview
4. Ensure logo is clear and meets requirements

### Step 6: Submit Verification Documents

CoinMarketCap may require:

#### Proof of Authenticity
- **Domain Verification**: Prove ownership of official website
- **Social Media Verification**: Verify official social accounts
- **Team Verification**: Provide team member information

#### Supporting Documents
- Project whitepaper or documentation
- Audit reports (if available)
- Legal opinion (for regulated projects)
- Proof of reserves (for stablecoins)

### Step 7: Review and Submit

1. Review all information for accuracy
2. Check all links are working
3. Verify contract address is correct (checksum format)
4. Agree to CoinMarketCap terms
5. Submit the application

### Step 8: Post-Submission

After submission:
1. You'll receive a confirmation email
2. Track application status in your CMC account
3. Respond promptly to any requests for additional information
4. Be patient - review can take several weeks

## ðŸ“Š What Happens After Submission

### Review Process
1. **Initial Review** (1-2 weeks): CMC team reviews submission
2. **Verification** (1-2 weeks): Team verifies project authenticity
3. **Quality Check** (varies): Assessment of project quality and legitimacy
4. **Approval/Rejection**: Decision communicated via email

### If Approved
- Token page created on CoinMarketCap
- Logo displayed on token page
- Price tracking begins
- Token appears in search results
- Historical data starts collecting
- Token listed in rankings (based on market cap)

### If Additional Info Needed
- CMC will email requesting specific information
- Respond within 7 days to avoid delays
- Provide clear, accurate information

## âš ï¸ Common Issues & Solutions

### Issue: Application Pending for Weeks
**Solutions:**
- Be patient - CMC reviews many applications
- Ensure all information was complete
- Check spam folder for CMC emails
- Can follow up after 4 weeks

### Issue: Verification Failed
**Solutions:**
- Verify domain ownership properly
- Ensure social media accounts are official
- Provide additional proof if requested
- Maintain consistent branding across platforms

### Issue: Insufficient Trading Activity
**Solutions:**
- Build liquidity on DEX/CEX
- Increase trading volume
- Add more trading pairs
- Reapply after establishing market presence

### Issue: Logo Rejected
**Solutions:**
- Ensure logo is exactly 200x200 pixels
- Use PNG format with transparency
- Provide high-quality, clear logo
- Avoid copyrighted or trademarked elements

### Issue: Missing Information
**Solutions:**
- Complete all required form fields
- Provide working links to all resources
- Include comprehensive project description
- Add all available trading pairs
