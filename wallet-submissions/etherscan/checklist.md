# Etherscan Logo Submission Checklist

Use this checklist to ensure your Etherscan logo submission is complete and meets all requirements.

## Pre-Submission Checklist

### Logo File Requirements
- [ ] Logo is PNG format
- [ ] Logo dimensions are at least 256x256px (512x512px recommended)
- [ ] File size is less than 200KB
- [ ] Background is transparent (alpha channel)
- [ ] Logo is clear and high quality
- [ ] Aspect ratio is 1:1 (square)
- [ ] Color space is sRGB

### Token Requirements
- [ ] Contract is deployed on Ethereum Mainnet
- [ ] Contract is verified on Etherscan
- [ ] Token is a standard ERC20/ERC721/ERC1155
- [ ] Token has on-chain activity (transactions)
- [ ] Contract address is in checksum format: `0x7BeB51807E3c8BdB10A2868bD51c2D9E1764925D`
